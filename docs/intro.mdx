# Build from source

```bash
sudo apt install build-essential
wget https://go.dev/dl/go1.26.3.linux-amd64.tar.gz
sudo rm -rf /usr/local/go && sudo tar -C /usr/local -xzf go1.26.3.linux-amd64.tar.gz

git clone https://github.com/Yayoi-cs/fastDbg
cd fastDbg
sh install.sh
```
